## What is the Notification Banner?

The Rancher dashboard includes a dynamic notification banner on the home page. It displays notifications about new releases, End of Maintenance (EOM) or End of Life (EOL) statuses, and other important announcements. This ensures that you stay informed about the latest updates and lifecycle events relevant to your deployment.

You can interact with these updates by clicking the **Learn More...** link on the right side of the banner or dismiss the notification using the **X** icon.
